### Verifizierte SEN0220-Fakten

```
Typ:                DFRobot SEN0220 — intern MH-Z16-Chip (NDIR)
Messbereich:        0–50000 ppm
Versorgung:         4.5–5.5V DC (KEIN 3.3V-Betrieb)
Durchschnittsstrom: < 85 mA
Schnittstelle:      UART, 9600 baud, SERIAL_8N1
Warmup-Zeit:        3 Minuten (erster Wert unzuverlaessig)
TX-Logic-Level:     3.3V-kompatibel (MH-Z16-Chip, bestaetigt via espboards.dev)
```

### UART-Protokoll (9-Byte Frame)

**Command "Read Gas Concentration":**
```
0xFF 0x01 0x86 0x00 0x00 0x00 0x00 0x00 0x79
```
Byte 0: Start (0xFF)
Byte 1: Sensor-Nr (0x01)
Byte 2: Command (0x86 = Read CO2)
Byte 3-7: Reserved (0x00)
Byte 8: Checksum (0x79 fuer diesen Command)

**Response (9 Bytes):**
```
Byte[0] = 0xFF  (Start)
Byte[1] = 0x86  (Command Echo)
Byte[2] = HIGH-Byte CO2-Wert
Byte[3] = LOW-Byte CO2-Wert
Byte[4..7] = Reserved / interne Daten
Byte[8] = Checksum
```
CO2-Wert in ppm = (Byte[2] << 8) + Byte[3]

**Checksum-Formel (Standard MH-Z16/MH-Z19):**
Checksum = (~(Byte[1] + Byte[2] + ... + Byte[7]) + 1) & 0xFF
(Einerkomplement der Summe Bytes 1-7, dann +1, masked auf 8 Bit)

### ESP32-UART-Tabelle (Standard-DevKit)

| UART | Default TX | Default RX | Status |
|------|-----------|-----------|--------|
| UART0 | GPIO 1 | GPIO 3 | UART0 = USB/Debug — NIEMALS fuer Sensoren |
| UART1 | GPIO 10 | GPIO 9 | GPIO9/10 = SPI-Flash-intern — NIEMALS nutzbar |
| UART2 | GPIO 17 | GPIO 16 | EMPFOHLEN fuer externe Sensoren |

**Robin hat GPIO16 (RX2) / GPIO17 (TX2) — das ist UART2, korrekt.**

### Level-Shifter-Analyse

- SEN0220 TX-Pegel: 3.3V-kompatibel → ESP32 RX direkt anschliessbar, kein Level-Shifter
- ESP32 TX (3.3V) → SEN0220 RX: 5V-Geraet akzeptiert >2.0V als HIGH → kein Level-Shifter noetig
- SEN0220 VCC: 5V → separate 5V-Leitung, KEIN 3.3V
- **Fazit: Hardware-Verkabelung von Robin (direkt, kein Level-Shifter) ist korrekt**
