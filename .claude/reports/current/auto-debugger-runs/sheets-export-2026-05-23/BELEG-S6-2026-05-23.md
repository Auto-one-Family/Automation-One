## D4 Granularitaets-Entscheidung

Optionen aus AUT-448:

| Option | Begruendung | Entscheidung |
|---|---|---|
| A: monatlich (`YYYY-MM`) | ~233k Zellen/Tag (Standard 9 Sensoren) -> ~7 Mio Zellen/Monat (43 Tage bis 10 Mio Limit). Akzeptabel mit Performance-Toleranz. | **Default** (ENV: `SHEETS_TAB_GRANULARITY=monthly`) |
| B: woechentlich (`YYYY-MM-W##`) | ISO-Woche, ~1,6 Mio Zellen/Woche. Performance-Sicher auch bei Multi-Value-Boom. | optional via ENV `SHEETS_TAB_GRANULARITY=weekly` |
| C: threshold-basiert | komplexer (State-Tracking), keine deterministische Tab-Namensgebung -> NICHT umgesetzt. | abgelehnt |

Begruendung Default: AUT-442 Parent rechnet mit 9 Sensoren + 30s Intervall. Bei Multi-Value-Boom (D4 16 Rows/min) ist Performance-Grenze in ~3 Tagen — der Operator kann via ENV-Switch auf weekly umstellen, ohne Code-Change. Backwards-Compatible.

## Tab-Namensgebung

```
SENSOR_TAB = f"sensoren-{YYYY}-{MM}"        # monthly default
SENSOR_TAB = f"sensoren-{YYYY}-{MM}-W{WW}"  # weekly (ISO 8601 week)
ACTUATOR_TAB = f"aktoren-{YYYY}-{MM}"
ACTUATOR_TAB = f"aktoren-{YYYY}-{MM}-W{WW}"
```

Zeitzone fuer Naming: **Europe/Berlin** (operator-orientiert). Daten innerhalb des Tabs bleiben UTC + lokal Doppelspalte.

## Initialisierung (Header-Idempotenz)

Beim ersten Schreiben in einen neuen Tab:

1. `worksheet_by_name(tab_name)` -> NotFound -> `add_worksheet(name, rows=1000, cols=20)`.
2. `append_row(HEADER_ROW)` mit `value_input_option=RAW`.
3. Erst dann den eigentlichen Datensatz appendend.

Bestehende Monats-Tabs bleiben unangetastet (kein Rolling-Delete, kein Re-Init).

## Guard gegen Mehrfach-Anlage

In-Memory-Cache `_known_tabs: set[str]` pro Service-Instanz + DB-Cache via `system_config` Key `sheets_export_known_tabs` (list of tab-names). Bei Restart wird Cache aus DB rehydriert.

```
_tab_lock = asyncio.Lock()
async def ensure_tab(name: str, header: list[str]) -> Worksheet:
    if name in _known_tabs:
        return self._client.worksheet(name)
    async with _tab_lock:
        if name in _known_tabs:  # double-check
            return ...
        ws = await self._client.get_or_create_worksheet(name, header)
        _known_tabs.add(name)
        await self._cursor.add_known_tab(name)
        return ws
```
