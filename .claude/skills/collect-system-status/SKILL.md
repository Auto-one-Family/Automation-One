# Skill: Collect System Status (From Code Perspective)

## Purpose
Analyze project structure, count files, parse configs → Send to Technical Manager.

## Execution Steps

### 1. Docker-Compose Analysis
```bash
# Parse docker-compose.yml
Read docker-compose.yml
Count services (expected: 8)
List profiles (expected: monitoring)
Extract all ports
```

### 2. Backend Structure (El Servador)
```bash
# Router count (CORRECTED: 18, not 14)
Glob "El Servador/god_kaiser_server/src/api/v1/*.py"
Count files excluding __init__.py

# Model count (CORRECTED: 17, not 16)
Glob "El Servador/god_kaiser_server/src/db/models/*.py"
Count files excluding __init__.py

# Service count
Glob "El Servador/god_kaiser_server/src/services/**/*.py"
Count files

# Test count
Glob "El Servador/god_kaiser_server/tests/**/*.py"
Count files
```

### 3. Frontend Structure (El Frontend)
```bash
# Component count
Glob "El Frontend/src/components/**/*.vue"
Count files

# View count
Glob "El Frontend/src/views/*.vue"
Count files

# Store count
Glob "El Frontend/src/stores/*.ts"
Count files

# API client count
Glob "El Frontend/src/api/*.ts"
Count files
```

### 4. ESP32 Firmware (El Trabajante)
```bash
# Source files
Glob "El Trabajante/src/**/*.cpp"
Glob "El Trabajante/src/**/*.h"
Count total

# PlatformIO environments
Read "El Trabajante/platformio.ini"
Extract [env:*] sections
```

### 5. Agent System (.claude/)
```bash
# Agents
Glob ".claude/agents/*/*.md"
Count directories

# Skills
Glob ".claude/skills/*/"
Count directories

# Current reports
Glob ".claude/reports/current/*.md"
List filenames
```

### 6. CI/CD (.github/workflows/)
```bash
# Workflows
Glob ".github/workflows/*.yml"
Count files

# Makefile targets (extract from Makefile)
Grep "^[a-zA-Z0-9_-]+:" Makefile
Count matches
```
